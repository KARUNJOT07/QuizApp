

## Project Description
This project is an Android Quiz App developed in Android Studio using Java and XML. The app allows users to enter their name, answer multiple-choice quiz questions, receive immediate visual feedback, track progress using a progress bar, and view their final score on a result screen.

 ## Features
- User name input on the main screen
- Multiple-choice quiz questions
- Correct answer shown in green
- Wrong selected answer shown in red
- User cannot change answer after submission
- Progress bar for quiz tracking
- Final score screen
- Take New Quiz button
- Finish button
- Dark Mode / Light Mode toggle
- Session persistence for saved user name and theme

## Tools Used
- Android Studio
- Java
- XML
- GitHub

## How to Run
1. Open the project in Android Studio
2. Allow Gradle sync
3. Run the app on an emulator or Android device
